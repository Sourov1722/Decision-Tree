## Decision Tree Comparison: CART vs ID3

This project implements and compares two Decision Tree algorithms:
- CART (Gini Index)
- ID3 (Entropy)

### Workflow
- Dataset loaded automatically from GitHub
- Missing value handling and categorical encoding
- Hyperparameter tuning using Cross Validation
- Evaluation using Accuracy, Precision, Recall, F1-score, and AUC
- Visualization of Confusion Matrix, ROC Curve, and Tree Structure

### Tools
- Python
- Scikit-learn
- Pandas
- Matplotlib
